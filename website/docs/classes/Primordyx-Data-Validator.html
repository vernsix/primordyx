Class Validator

A utility class to validate associative arrays against field rules. Supports rule strings (e.g., "required|email|min:18") or callables for custom logic. Returns an array of error messages per field â€” does not throw exceptions.

Performs validation on form or data arrays using string-based rule syntax or callables. String rules are separated by pipe (|), and support parameters via colon (e.g., min:18).
